# SWOT-СИСТЕМА: 10 промтов

## Каталог промтов

| # | Название | Суть | Время |
|---|----------|------|-------|
| P1 | СТРАТЕГ-360 | Полный SWOT + стратегии за один запрос | 25 мин |
| P2 | РЕНТГЕН СИЛЫ | Глубокий анализ сил через VRIO-фильтр | 20 мин |
| P3 | ДЕТЕКТОР ДЫРЫ | Честный аудит слабостей с приоритизацией | 20 мин |
| P4 | РАДАР СРЕДЫ | Сканирование внешней среды (PESTEL + Портер) | 20 мин |
| P5 | МОСТ К СТРАТЕГИИ | TOWS-матрица: из анализа в конкретные действия | 20 мин |
| P6 | АНТИБУЛЛШИТ | Проверка готового SWOT через 6 тестов | 15 мин |
| P7 | ШАХМАТНАЯ ДОСКА | Сравнительный SWOT по конкурентам | 25 мин |
| P8 | ЛАКМУС | Экспресс-оценка идеи за 10 минут | 10 мин |
| P9 | КАРЬЕРНЫЙ КОМПАС | Персональный SWOT для карьеры | 15 мин |
| P10 | ПУЛЬС СТРАТЕГИИ | Квартальный апдейт существующего SWOT | 15 мин |

---

## Три режима работы

**Спринт (15 мин)** — новая идея, быстрая оценка:
```
ЛАКМУС → решение go/no-go
```

**Полный цикл (2-3 часа)** — стратегическое планирование:
```
РАДАР СРЕДЫ → РЕНТГЕН СИЛЫ → ДЕТЕКТОР ДЫРЫ → СТРАТЕГ-360 → АНТИБУЛЛШИТ → МОСТ К СТРАТЕГИИ → ШАХМАТНАЯ ДОСКА
```

**Мониторинг (30 мин)** — раз в квартал:
```
ПУЛЬС СТРАТЕГИИ → АНТИБУЛЛШИТ
```

---

## Как пользоваться

**Шаг 0.** Заполни досье компании (шаблон ниже). Это 50% успеха.

**Шаг 1.** Выбери режим работы.

**Шаг 2.** Запускай промты по цепочке. Результат каждого промта — вход для следующего.

**Шаг 3.** Всегда прогоняй через АНТИБУЛЛШИТ перед финализацией.

**Шаг 4.** Обязательно доходи до МОСТА К СТРАТЕГИИ. SWOT без стратегий — бесполезен.

**Три правила:**
- Чем подробнее вводные — тем точнее результат
- Каждый промт питается результатами предыдущего — не перескакивай
- После каждого промта добавляй свою экспертизу и корректируй

## Адаптация под задачу

| Задача | Цепочка промтов |
|--------|-----------------|
| Стартап | ЛАКМУС → РАДАР → СТРАТЕГ-360 → АНТИБУЛЛШИТ |
| Зрелая компания | Полный цикл, акцент на слабости и угрозы |
| Новый рынок | РАДАР → ЛАКМУС → СТРАТЕГ-360 → ШАХМАТНАЯ ДОСКА → МОСТ |
| Привлечение инвестиций | РЕНТГЕН → РАДАР → СТРАТЕГ-360 → АНТИБУЛЛШИТ → МОСТ |
| Антикризис | ДЕТЕКТОР → РАДАР → СТРАТЕГ-360 → МОСТ (фокус на WT/ST) |
| Карьера | КАРЬЕРНЫЙ КОМПАС |

---

## Частота

- **Раз в год** — полный цикл (3-4 часа)
- **Раз в квартал** — мониторинг (30 мин)
- **По событию** — при резких изменениях
- **Для новых идей** — ЛАКМУС (10 мин)

---

## Типичные ошибки

1. Запускать СТРАТЕГ-360 без подготовки данных — получишь generic
2. Пропускать АНТИБУЛЛШИТ — рискуешь строить стратегию на иллюзиях
3. Останавливаться на табличке — без TOWS-стратегий SWOT бесполезен
4. Пихать всё в один промт — несколько точечных запросов дают лучший результат
5. Не обновлять — SWOT устаревает за 2-3 месяца
6. Не добавлять свою экспертизу — ИИ не знает вашей внутренней кухни

## Промты

### 1. СТРАТЕГ-360

```
Роль: Стратегический консультант уровня McKinsey/BCG.

Проведи глубокий SWOT-анализ.

Компания/проект: [НАЗВАНИЕ]
Отрасль: [ОТРАСЛЬ]
Стратегическая цель: [ЦЕЛЬ]
Контекст: [ОПИСАНИЕ СИТУАЦИИ]

Требования:

1. СИЛЬНЫЕ СТОРОНЫ (5-8 пунктов, внутренние):
   - Конкретное проявление + почему это преимущество относительно цели
   - Устойчивость (легко ли скопировать)

2. СЛАБЫЕ СТОРОНЫ (5-8, внутренние):
   - Как ограничивает движение к цели
   - Критичность (высокая/средняя/низкая) + возможность устранения

3. ВОЗМОЖНОСТИ (5-8, внешние):
   - Временное окно (срочная/среднесрочная/долгосрочная)
   - Вероятность и потенциальный эффект

4. УГРОЗЫ (5-8, внешние):
   - Вероятность и масштаб ущерба
   - Скорость наступления

5. ПЕРЕКРЁСТНЫЙ АНАЛИЗ (TOWS):
   - SO-стратегии: минимум 3
   - WO-стратегии: минимум 3
   - ST-стратегии: минимум 3
   - WT-стратегии: минимум 2

6. ТОП-3 приоритета с первым конкретным шагом для каждого.

Каждый пункт — конкретный и специфичный для этой компании, не generic.
```

### 2. РЕНТГЕН СИЛЫ

```
Роль: Аналитик конкурентных преимуществ.

Компания: [НАЗВАНИЕ]
Отрасль: [ОТРАСЛЬ]
Конкуренты: [КОНКУРЕНТ 1, 2, 3]

Проанализируй сильные стороны по категориям:

1. Ресурсные (финансы, люди, активы, нематериальные активы)
2. Компетентностные (что делаем лучше конкурентов, уникальные процессы)
3. Позиционные (доля рынка, отношения с клиентами, репутация)

Для каждой силы — VRIO-анализ:
- Ценно? (даёт преимущество на рынке)
- Редко? (есть ли у конкурентов)
- Сложно скопировать? (время/деньги для конкурента)
- Используется? (извлекается ли реальная польза)

Ранжируй: устойчивое преимущество → временное → паритет с рынком.
```

### 3. ДЕТЕКТОР ДЫРЫ

```
Роль: Кризис-менеджер и внутренний аудитор.

Компания: [НАЗВАНИЕ]
Отрасль: [ОТРАСЛЬ]
Болевые точки (если известны): [ОПИСАНИЕ]

Безжалостно честный анализ слабостей:

1. Операционные (процессы, узкие места, качество, bus factor)
2. Финансовые (cash flow, издержки, зависимость от одного источника дохода)
3. Рыночные (бренд, клиентская база, каналы, отставание от конкурентов)
4. Организационные (культура, управление, компетенции, технологии)

Для каждой слабости:
- Конкретное проявление
- Влияние на бизнес (1-10)
- Сложность устранения (1-10)
- Вердикт: ИСПРАВИТЬ / КОМПЕНСИРОВАТЬ / ПРИНЯТЬ

Не маскируй слабости под «зоны роста».
```

### 4. РАДАР СРЕДЫ

```
Роль: Аналитик внешней среды и трендов.

Компания: [НАЗВАНИЕ]
Отрасль: [ОТРАСЛЬ]
Рынок: [ГЕОГРАФИЯ]
Горизонт: [1/3/5 лет]

Анализ через PESTEL (Political, Economic, Social, Technological, Environmental, Legal):
- По каждому фактору: возможности и угрозы

Анализ через 5 сил Портера:
- Новые игроки, заменители, сила поставщиков, сила покупателей, конкуренция

Для каждой возможности: временное окно, вероятность, эффект.
Для каждой угрозы: скорость наступления, вероятность, масштаб ущерба.

Итог: что требует реакции СЕЙЧАС / что мониторить / что игнорировать.
```

### 5. МОСТ К СТРАТЕГИИ

```
Роль: Стратегический консультант.

Входные данные:
S: [S1, S2, S3...]
W: [W1, W2, W3...]
O: [O1, O2, O3...]
T: [T1, T2, T3...]

Построй TOWS-матрицу:

1. SO-стратегии («Атака»): используем силы для захвата возможностей
2. WO-стратегии («Подтягивание»): преодолеваем слабости ради возможностей
3. ST-стратегии («Защита»): используем силы для нейтрализации угроз
4. WT-стратегии («Выживание»): минимизируем слабости и избегаем угроз

Каждая стратегия: «Использовать/Устранить [X] для [Y] через [конкретный шаг]»

Приоритизация по: воздействие (1-10) × реалистичность (1-10).

ТОП-5 стратегий с:
- Формулировкой (1 предложение)
- 3 первыми шагами
- KPI
- Рисками при реализации
```

### 6. АНТИБУЛЛШИТ

```
Роль: Дьявольский адвокат и критический рецензент.

Вот SWOT-анализ: [ВСТАВИТЬ]

Проверь по 6 тестам:

1. Специфичность — пункт применим только к этой компании или к любой?
2. Категоризация — внешнее не попало в S/W? Внутреннее не попало в O/T?
3. Обоснованность — что основано на фактах, что на ощущениях?
4. Полнота — что пропущено? Какие неудобные правды замолчаны?
5. Актуальность — всё ещё релевантно?
6. Actionability — можно ли на основе каждого пункта принять решение?

Выдай: оценку (1-10), список исправлений, пропущенное, улучшенную версию.
```

### 7. ШАХМАТНАЯ ДОСКА

```
Роль: Конкурентный аналитик.

Моя компания: [ОПИСАНИЕ]
Конкурент 1: [ОПИСАНИЕ]
Конкурент 2: [ОПИСАНИЕ]
Конкурент 3: [ОПИСАНИЕ]

1. Краткий SWOT для каждой компании
2. Сравнительная таблица сил (оценка 1-5 по критериям)
3. Карта уязвимостей конкурентов + как использовать
4. Карта наших уязвимостей + как конкуренты могут использовать
5. Рекомендации: где атаковать, где защищаться, где дифференцироваться
```

### 8. ЛАКМУС

```
Роль: Практичный бизнес-ментор.

Идея: [2-3 ПРЕДЛОЖЕНИЯ]
Целевой рынок: [КТО КЛИЕНТ]
Что уже есть: [РЕСУРСЫ, ОПЫТ]

Экспресс-SWOT:
- ТОП-3 силы (почему взлетит)
- ТОП-3 слабости (почему НЕ взлетит)
- ТОП-3 возможности (попутные ветры)
- ТОП-3 угрозы (что может убить)

Каждый пункт — одно конкретное предложение.

Плюс:
- Главный вопрос перед запуском
- Самый важный эксперимент
- Kill-criteria: при каких условиях отказаться от идеи
```

### 9. КАРЬЕРНЫЙ КОМПАС

```
Роль: Карьерный стратег и executive coach.

Текущая роль: [ДОЛЖНОСТЬ]
Опыт: [ГОДЫ, ОБЛАСТИ]
Навыки: [КЛЮЧЕВЫЕ]
Карьерная цель: [КУДА ЧЕРЕЗ 2-3 ГОДА]
Индустрия: [ОТРАСЛЬ]

Персональный SWOT:
- Strengths: hard skills, soft skills, уникальный опыт, нетворк
- Weaknesses: недостающие навыки, пробелы в опыте, ограничивающие паттерны
- Opportunities: тренды рынка труда, растущие области, ресурсы для развития
- Threats: автоматизация/AI, конкуренция, изменения в индустрии

План:
- 3 навыка прокачать за 6 месяцев
- 3 действия к цели
- 1 вещь перестать делать
```

### 10. ПУЛЬС СТРАТЕГИИ

```
Роль: Стратегический аналитик.

Предыдущий SWOT (дата: [ДАТА]):
[ВСТАВИТЬ]

Изменения с тех пор:
[ОПИСАНИЕ]

Задача:
1. Аудит: каждый пункт — усилился/ослаб/без изменений/исчез
2. Новые элементы: новые S, W, O, T
3. Переквалификация: что поменяло категорию (возможность→угроза и т.д.)
4. Обновлённый SWOT с пометками изменений
5. Вывод: нужно ли менять стратегию, что корректировать срочно
```
